About The Project

An advanced Content Management System (CMS) designed using Python Flask, Jinja2, and SQLAlchemy as a final year model project showcase.

Project Abstract

The principal objective of this project is to model and implement a production-ready, fully responsive Content Management System (CMS) using a classical Model-View-Controller (MVC) architecture pattern. The application enables users to view published articles, filter posts based on dynamic category definitions, and engage with content authors through an inline AJAX-driven commenting engine without experiencing full page reloads.

On the administrative spectrum, the system features a dedicated, secured Dashboard control center. Authorized administrators receive real-time analytical metrics mapping global counts (posts, categories, users) and dynamic graphical distributions powered by Chart.js. Full CRUD operations enable admins to add, edit, and delete published/draft articles, assign SEO-friendly URL slugs, and manage global post categories dynamically.

Key Features

  • Secure Authentication: Password hashing using PBKDF2/SHA256 and Session tracking via Flask-Login.
  • Interactive Analytics: Doughnut graphs plotting posts density using Chart.js.
  • Media Management: Image validation uploads for profile pic and articles.
  • SEO URLs: Automated string transliterations to construct elegant URL slugs.
  • Modular UI: Dual-mode transitions (Dark & Light themes) leveraging native browser local storage persistence.
